## ⚠️ 铁律与注意事项

| 规则 | 详情 |
|------|------|
| **09:20 盘前时间** | 不可改为 09:15 或其他 |
| **禁止折叠** | 所有卡片始终展开；无数据时显示占位提示 |
| **红涨绿跌** | A股惯例，不可颠倒 |
| **部署前检查** | sync_check.py 自动检查 CLOSED_SET 防御 + neodata双查询标记 |
| **neodata token** | JSON格式 `{"token": "tk_...", "saved_at": ...}`，代码必须 `json.load()` 取 `.get("token")` |
| **停更数据源** | 主力个股/北向资金 仅周一执行定时任务 |
| **卡片总结语句** | 永远拼入渲染函数的 `h` 字符串中，不要放在 HTML 模板的 content 容器内部（会被 innerHTML 覆盖） |
