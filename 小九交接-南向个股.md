# 小九交接 — 南向资金个股明细

## 背景
南向资金个股明细（流入/流出 TOP5）需要调用东方财富 `push2.eastmoney.com` 接口。
**阿狸咪家 IP 目前被该接口封禁**，每次获取都返回空。

## 解决方案
独立脚本 `fetch_south_individual.py` 已写好。

## 小九操作步骤

### 每天收盘后（或任意时间）

1. **先 git pull 拉取最新代码和数据**
   ```
   cd E:/workspace/stock-scanner
   git pull origin main
   ```

2. **运行南向个股脚本**
   ```
   python fetch_south_individual.py
   ```
   
3. **成功 → 注入数据并部署**
   ```
   python update_data_v2.py --fast
   python deploy_now.py --force
   ```
   
4. **失败 → 跳过，等下次**
   - 脚本会打印 `⚠️ 获取失败（可能是IP被封）`
   - 不会修改任何数据文件
   - 前端南向个股块自动隐藏，不影响其他功能

### 🎯 判断标准
- ✅ 成功：显示 "流入X只 流出X只"
- ❌ 失败：显示 "RemoteDisconnected" 或 "NoneType"

## 注意事项
- 如果小九单位网络也失败了，说明东方财富彻底改了接口，需要我来换数据源
- 成功获取的数据已包含在当天部署中，阿狸咪无需再跑
- 这个接口的封禁通常是**暂时性**的，过几小时~几天可能就通了
