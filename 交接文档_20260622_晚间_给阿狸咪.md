## ⚠️ 部署注意事项

`update_data_v2.py` 在采集宏观数据时会卡在网络请求上（Binance/CoinGecko 比特币 API 超时），导致部署被阻塞。

**临时解决方案**（本次使用）：
```bash
# 1. 重命名 update_data_v2.py 使其找不到
mv update_data_v2.py update_data_v2.py._temp
# 2. 直接部署（使用已有 dist/）
python deploy_now.py --force
# 3. 恢复
mv update_data_v2.py._temp update_data_v2.py
```

> 因为 code 改动物理上只影响 `index_master.html`，dist/ 里的旧 HTML 也需要同步代码改动。建议以后：
> - 小改动直接用上面方法，跳过宏观数据采集
> - 大改动等网络正常时跑完整流程
